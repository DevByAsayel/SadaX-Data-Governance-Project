# SadaX – Data Governance Project

This project presents a data governance framework for **SadaX**, a secure encrypted communication network designed to provide trusted messaging and communication services.

## Project Overview
The project focuses on:
- Strategic framework for SadaX
- Data classification process
- Data privacy process
- Regulatory and compliance alignment
- Data dictionary and ERD
- Data inventory
- Risk analysis using FMEA
- Data glossary

## Key Topics Covered
- End-to-End Encryption (E2EE)
- Role-Based Access Control (RBAC)
- Personal Data Protection Law (PDPL)
- National Cybersecurity Authority (NCA) controls
- SAMA cybersecurity considerations
- ARAMCO CCC alignment
- Data governance policies and standards
